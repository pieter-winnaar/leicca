# ING Me Font Files

## Required Files

Place the ING Me font files in this directory:

### Regular (400 weight)
- `INGMe-Regular.woff2` (preferred - best compression)
- `INGMe-Regular.woff` (fallback)

### Bold (700 weight)
- `INGMe-Bold.woff2` (preferred)
- `INGMe-Bold.woff` (fallback)

## Where to Download

**Option A: OnlineWebFonts.com**
1. Regular: https://www.onlinewebfonts.com/download/69c633b2a4e41e8101c6f4f149655d5e
2. Bold: https://www.onlinewebfonts.com/download/5724ce1882e134bfbb68a761b8e85b51

**Option B: WebFontFree.com**
1. Regular: https://www.webfontfree.com/en/download/INGMe-Regular
2. Bold: https://www.webfontfree.com/en/download/INGMe-Bold

## Installation Steps

1. Download the font packages from one of the sources above
2. Extract the `.woff2` and `.woff` files
3. Rename files to match the expected names above
4. Place files in this directory
5. The fonts will be automatically loaded via `globals.css`
